BLM Idaho Range Improvement Point

The Bureau of Land Management, housed within the US Department of the Interior, manages features that help sustain the health, diversity, and productivity of public lands. This map shows a subset of the features managed by the BLM in Idaho, color-coded by type of feature.


Current functionality includes:

  • Color coding by feature type
  • Clicking on an marker shows the feature name and type
  • Checkboxes filters which markers show on the map
